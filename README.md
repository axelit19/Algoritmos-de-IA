# Algoritmos de IA

En este espacio pondré algunos colab acerca de algoritmos de IA.

## Contenido

Este repositorio contiene implementaciones y notebooks de Google Colab sobre distintos algoritmos de Inteligencia Artificial, incluyendo:

- **Búsqueda y optimización**: BFS, DFS, A*, algoritmos genéticos
- **Aprendizaje automático**: regresión, clasificación, clustering
- **Redes neuronales**: perceptrón, backpropagation
- **Lógica y agentes**: sistemas expertos, agentes reactivos

## Cómo usar

Los notebooks de Google Colab se pueden abrir directamente desde GitHub haciendo clic en el archivo `.ipynb` y luego seleccionando "Abrir en Colab".

Los scripts de Python se pueden ejecutar localmente con:

```bash
python nombre_del_script.py
```

## Algoritmos incluidos

### Búsqueda

| Algoritmo | Descripción | Archivo |
|-----------|-------------|---------|
| BFS | Búsqueda en anchura | `busqueda/bfs.py` |
| DFS | Búsqueda en profundidad | `busqueda/dfs.py` |
| A* | Búsqueda heurística | `busqueda/a_estrella.py` |

### Aprendizaje Automático

| Algoritmo | Descripción | Archivo |
|-----------|-------------|---------|
| Regresión Lineal | Predicción con variables continuas | `ml/regresion_lineal.py` |
| K-Means | Clustering no supervisado | `ml/kmeans.py` |
| Árbol de Decisión | Clasificación supervisada | `ml/arbol_decision.py` |
